# 🏡 House-Price-Prediction

Predicting house prices is a real-world problem where multiple factors influence the final value of a property. In this project, I developed a machine learning model to estimate house prices based on different housing features and property characteristics.

## 📌 Project Objective

The main goal of this project was to analyze housing data, identify the factors affecting property prices, and build a machine learning model capable of making reliable price predictions.

## 📊 Step 1: Understanding the Dataset

The dataset contains information related to residential properties, including:

* 📐 Area of the house
* 🛏️ Number of bedrooms
* 🚿 Number of bathrooms
* 🏢 Number of stories
* 🚗 Parking availability
* ❄️ Air conditioning
* 📍 Preferred location
* 🛋️ Furnishing status
* 🏠 Additional housing amenities

Understanding these features helped in identifying the variables that influence house prices the most.

## 🧹 Step 2: Data Cleaning & Preprocessing

To prepare the dataset for machine learning:

* Checked for missing values
* Removed duplicate records
* Verified data consistency
* Converted categorical features into numerical format
* Prepared the dataset for model training

## 📈 Step 3: Exploratory Data Analysis (EDA)

Performed EDA to gain insights into the dataset by:

* Analyzing feature distributions
* Identifying trends and patterns
* Exploring relationships between variables
* Creating visualizations for better understanding

This step helped uncover important factors affecting housing prices.

## 🔍 Step 4: Feature Analysis

A detailed analysis was performed to understand the impact of each feature on house prices.

Key factors analyzed:

* 📐 Property Area
* 🛏️ Bedrooms
* 🚿 Bathrooms
* 🏢 Stories
* 🚗 Parking Spaces
* ❄️ Air Conditioning
* 📍 Location Preference
* 🛋️ Furnishing Status

Correlation analysis was also used to study relationships between numerical variables.

## 🤖 Step 5: Model Development

After preprocessing the data:

* Split the dataset into training and testing sets
* Trained a machine learning regression model
* Learned patterns from historical housing data
* Generated price predictions based on property features

## 📉 Step 6: Model Evaluation

The model was evaluated using unseen test data to measure its performance.

Evaluation focused on:

* Prediction accuracy
* Error analysis
* Model reliability
* Generalization on unseen data
